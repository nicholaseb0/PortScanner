#  PortScanner

A simple multithreaded TCP port scanner built in Python.


 

-  Fast multithreaded scanning
-  Custom port range support
-  Optional export to text file
-  Command-line interface (CLI)
-  Graceful error handling

---

##  Requirements

- Python 3.6+
